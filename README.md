# armv7_bootstrap
QEMU ARMv7 bare metal examples aimed at bootstrapping an OS kernel

## building and running
  - build: `make`
  - run an example: `./run.sh 00_helloworld/kernel.elf`
  - debug a running example: `./debugger.sh 00_helloworld/kernel.elf`
